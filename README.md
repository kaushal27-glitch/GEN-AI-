# AI|ML-learning-
opensource AI ML Learning model
A production-ready FastAPI application that:


Ingests research papers/documents (PDFs, TXT, docx)
Extracts and chunks documents intelligently
Creates vector embeddings using OpenAI/Ollama
Stores embeddings in ChromaDB (local) or Pinecone (cloud)
Answers questions with source attribution and citations
Compares naive LLM vs RAG-powered answers with accuracy metrics
